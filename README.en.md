# SpringBoot-Vue-042springboot-IT技术交流和分享平台

#### Description
基于Spring Boot 2.7+ + MyBatis Plus + Shiro的IT技术交流平台，包含前后端分离架构，实现笔记分享、收藏管理、权限控制等功能。后端采用三层架构设计，前端提供管理后台和用户门户界面，支持技术文档上传、互动讨论和用户行为分析。
